# Binary File Modifier (XOR)

Тестовое задание: программа модифицирует входные файлы бинарной операцией XOR
с заданным 8-байтным значением.

Qt Widgets, C++17, Windows, MinGW, сборка через CMake (`CMakeLists.txt`).

## Возможности

- **a)** Маска входных файлов (несколько масок через `;`, например `*.bin;*.txt;testFile.bin`)
- **b)** Опциональное удаление входных файлов после обработки
- **c)** Путь для сохранения результатов
- **d)** Путь для поиска входных файлов
- **e)** Действие при повторении имени выходного файла: перезапись или
  модификация имени (счётчик `name_1.ext`, `name_2.ext`, …)
- **f)** Режим работы: разовый запуск или работа по таймеру
- **g)** Периодичность опроса папки на наличие новых файлов
- **h)** 8-байтное значение для XOR в hex-формате (например `1234567890ABCDEF`)
  с кнопкой проверки

Дополнительно:

- Обработка файлов большого объёма (10+ GB) без «замирания» интерфейса:
  файл читается поблочно (1 МБ), обработка идёт в отдельном потоке
  (`FileProcessor`, `moveToThread`), связь с GUI — только через сигналы/слоты.
- Статус и прогресс: таблица файлов (размер, статус, прогресс, скорость,
  прошедшее и оставшееся время), панель деталей текущего файла, общий прогресс,
  журнал событий.
- Пауза и возобновление обработки в любой момент с того же места
  (атомарный флаг + `QWaitCondition`, позиция в файле сохраняется).
- Корректное закрытие программы даже во время обработки: в `closeEvent`
  потоку подаётся сигнал остановки, снимается пауза, затем
  `thread.quit(); thread.wait()`.
- XOR-ключ накладывается циклически с учётом смещения в файле, «хвост»
  файла, не кратный 8 байтам, обрабатывается корректно (повторный XOR
  восстанавливает исходный файл).

## Сборка

Qt 6.x (MinGW 64-bit), CMake, Ninja — например, из состава Qt:

```bat
cmake -S . -B build -G Ninja -DCMAKE_PREFIX_PATH=C:/Qt/6.11.1/mingw_64 -DCMAKE_BUILD_TYPE=Release
cmake --build build
build\kolibtitest.exe
```

Либо просто открыть `CMakeLists.txt` в Qt Creator и собрать комплектом
Desktop Qt MinGW 64-bit.

Если у вас не получилось, то феечки винкс помогут!
